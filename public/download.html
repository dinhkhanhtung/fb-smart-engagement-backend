🔧 Hướng dẫn chi tiết

Chrome/Edge (Chromium-based)
  1. Tải file FB-Smart-Engagement-Pro-Chrome.zip
  2. Giải nén file vào thư mục trên máy tính
  3. Mở Chrome/Edge, gõ: chrome://extensions/
  4. Bật "Developer mode" (góc trên bên phải)
  5. Click "Load unpacked"
  6. Chọn thư mục chrome/ đã giải nén
  7. Extension sẽ xuất hiện trong danh sách
Mozilla Firefox
  1. Tải file FB-Smart-Engagement-Pro-Firefox.zip
  2. Giải nén file vào thư mục trên máy tính
  3. Mở Firefox, gõ: about:debugging
  4. Click "This Firefox" trong menu bên trái
  5. Click "Load Temporary Add-on..."
  6. Chọn file firefox/manifest.json trong thư mục đã giải nén
  7. Extension sẽ xuất hiện trong danh sách
Lưu ý cho Firefox: Extension sẽ hoạt động ở chế độ tạm thời. Sau khi đóng Firefox, bạn cần tải lại extension.
